Installation intructions:

  1. Complete video installation instructions available at: this youtube video.
  2. Install the splash application.
  3. Download Winter's Kiss theme.
  4. Go to the Downloads folder of your PC and find Winters-Kiss.zip file.
  5. Extract the file in the Documents folder of your PC under the splash/themes folder.
  6. If splash/themes folder under the Documents folder does not exist then create one and extract over there.
  7. Now with the splash application running go to the system tray (where audio and battery icons are in the taskbar).
  8. There you will find the splash icon, right click on it would reveal a menu.
  9. From the menu, choose the 'Load File' option.
  10. Your file explorer will show up, locate the wintersKiss.html file under the extracted folder and click on open (option at the bottom of file explorer).
  11. Your selected file would render
